
Running the Program:								1.Navigate to the src folder in your terminal	 
	2.Run the labyrinthServer.py file in the python		  	interpreter using the commandline arguments:			  	python labyrinthServer.py <mapname> <gamemode>		 	3.Open a new terminal instance and repeat step one 	 	4.Run the labyrinth.py file in the python 				interpreter to solve the labyrinthmap you			   	have selected
